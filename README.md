 CryptoFlux 🪙

Trabajo Final - Programación III

Sistema de gestión de cartera de criptomonedas.

Tecnologías
- Frontend: Vue.js 3 + Vite + Axios
- Backend: ASP.NET Core 8 (C#)
- Base de datos: SQL Server
- ORM: Entity Framework Core

Cómo correr el proyecto

Backend
1. Abrí la solución `CryptoFlux/CryptoFlux.API` en Visual Studio 2022
2. Ejecutá el proyecto con F5
3. Swagger disponible en: `https://localhost:7212/swagger`

Frontend
1. Abrí la carpeta `crypto-flux` en VS Code
2. Ejecutá en la terminal:
```bash
npm install
npm run dev
```
3. La app estará disponible en: `http://localhost:5173`

Funcionalidades
-  Registrar compras de criptomonedas
-  Historial de movimientos
-  Ver, editar y eliminar transacciones
-  Base de datos persistente
